Class sandy.util.Sandy

sandy.util.Sandy

Description

Sandy An utility class which goal is to simplify the creation of Sandy's applications. It creates a camera , a screen and a fps counter. To use it you have to extend your application class (if you decided to work with classes) from this one and to create a createScene method that has the following prototype: youApp.createScene( bg:BranchGroup ):Void;

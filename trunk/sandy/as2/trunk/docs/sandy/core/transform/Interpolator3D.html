Interface sandy.core.transform.Interpolator3D

  • sandy.core.transform.ITransform3D
    • sandy.core.transform.Interpolator3D

Description

Represents a Transformation Interpolator for an Group.

The Interpolator3D is applied to each Object3D into a Group.
